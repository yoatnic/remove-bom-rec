# remove-bom-rec

Strip BOM recurcive.

## Getting Start

```sh
npm i
```

## Usage

```sh
node index.js -d path/to/directory -e cs,md,conf
```

#### Options

| Option   | Description                                                             |
| -------- | ----------------------------------------------------------------------- |
| -dir, -d | Target direcory.                                                        |
| -ext, -e | Target file extension. You can specify multi extesions(e.g. cs,md,conf) |
